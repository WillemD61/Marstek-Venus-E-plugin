# Marstek-Venus-E-plugin
specifically for Venus E testing

# this is not a production version yet, still under development (although it works fine on my Marstek Venus A and Raspberry Pi 3)

This is the Domoticz plugin as already available in "Marstek-Venus-plugin" for Domoticz, but now adapted to use the API from https://github.com/jaapp/ha-marstek-local-api to check whether this makes any difference for the communication to the Venus E.

please put these 3 files into the folder created by the regular Marstek-venus-plugin, restart domoticz and activate the plugin.

Please report any ssues here on github.
